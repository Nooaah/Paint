
###### PAINT VERSION 2.0 ######

Projet C : Réalisation d'un Paint

Réalisation : 	Noah CHATELAIN

####### EXECUTER PAINT ########

Dans le terminal : 

1)   chmod 777 paint
2) ./paint

######## RACCOURCIS ###########

1 : Rectangle
2 : Droite
3 : Cercle
4 : Retour en arrière
5 : Avancer
6 : Faire apparaître une nouvelle feuille
7 : Sauvegarder Liste dans le fichier Sauvegarde.c
8 : Grossir le trait de la dernière figure
Clic droit : Zoomer
